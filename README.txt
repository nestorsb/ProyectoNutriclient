*************************PARA LA PUESTA EN FUNCIONAMIENTO*************************************

    - Si no esta instalado, instalar node.js desde https://nodejs.org/es/download/, 
        se puede comprobar si esta instalado con  "node -v"

    -Ir a la carpeta raiz del proyecto
        - "cd .\frontend\" 
        - "npm run serve"
    
    -Abrir otra consola para no cerrar la ejecucion del comando anterior.
    -Ir a la carpeta raiz del proyecto
        - "cd .\Proyecto\" 
        - "npm run start"
    (El back debe estar en el puerto :9000), pues las consultas del Front apuntan a este puerto.

    Recuerda tener la BBDD Mysql importada del proyecto encendida y funcionando.
